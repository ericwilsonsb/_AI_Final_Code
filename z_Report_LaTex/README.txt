NO CLASSIFIED MATERIALS ARE ALLOWED. For additional guidance on using the 3 provided templates, please go to https://www.gomactech.net and review the "Information Markings Guidance: Markings for GOMACTech Papers" document.

GOMACTech uses the standard IEEE Conference LaTeX format with minor tweaks in the provided GOMACTech_LaTeX*.tex templates/examples to address required markings.

GOMATech_LaTeX*.tex require IEEEtran.cls, IEEEtran.bst, GOMACTech_LaTeX.bib and IEEEabrv.bib in order to compile correctly.

A how-to for the IEEEtran class along with several other helpful files can be found at the IEEE.org website.

********************************** Files **********************************

README.txt             - This file.

GOMACTech_LaTeX_Publicly_Approved.tex - The template/example file for GOMACTech papers have ALREADY BEEN APPROVED for public release.

GOMACTech_LaTeX_CUI_CTI.tex - The template/example file for CUI with Critical Technology Information (CTI).

GOMACTech_LaTeX_CUI_CTI_EXPT_EAR_ITAR.tex - The template/example file for CUI with CTI and Export-controlled information (either EAR or ITAR).

IEEEtran.cls           - The IEEEtran LaTeX class file.

IEEEtran.bst           - The IEEE Bibliography Style file.

GOMACTech_LaTeX.bib     - A bibliography file used in the GOMACTech_LaTeX.tex example.

IEEEabrv.bib           - A bibliography file used in the GOMACTech_LaTeX.tex example.
